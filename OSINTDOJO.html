02 - Instaquiz com by SECTOR035

This is the resolution of the 02 - Instaquiz by SECTOR035

The SECTOR035 OSINT Quiz can be find and practice by sending an email to osintquiz@gmail.com with the subject "start"

After that we are given a number of quiz to complete.

In this case we are going to solve the quiz number 08 - Instaquiz

The question of this quiz is "What is the profile id of the following Instagram account? https://www.instagram.com/micro_bar " and we need to find the anwser for this quiz and submit the key in md5 value to the email osintquiz@gmail.com with the subject of the hash value that we are going to generate.

Steps to reproduce:

- Login into your socket puppet or researcher account;

- Open the page that we want to search the id, in this case the page for this quiz it's https://www.instagram.com/micro_bar

Go to to the view source code of the page look for profilePage_ (use control + f to help find more quickly the profilePage_ value)

- In the source code it will show up something like this:profilePage_2260998159

- The value of 2260998159 is the awnser to the quiz

- Then hash the value that you have found (2260998159) using any tool that you want. Example https://www.md5hashgenerator.com/

- Then just send an email to osintquiz@gmail.com with the value in md5: 8ed3b525f18b66b4103c93b3d766786c

- To see if the hash value is the correct one for this quiz wait for the automatic response from the email and in the response you will see the same hash value that we just send.
